JTS GO

A Python-like programming language with a bytecode virtual machine. Simple syntax, fast execution, portable across platforms.

Stable v1.0.0 Windows + Linux Apache 2.0 License
$ npm install -g @jaytechsolutions/jts-go [ copy ]

Why JTS GO?

⚡

Python-like Syntax

Clean, readable code with indentation-based blocks, no braces. Write less, understand more.

⚙

Bytecode VM

Compiled to bytecode and executed on a stack-based virtual machine for fast, efficient execution.

⚖

Dynamic Typing

Number, boolean, string, nil, functions, and lists. No type declarations needed.

⚙

Native Functions

Built-in print(), input(), len(), type() functions ready to use.

☁

One Command Install

Install globally with npm. No compilation, no dependencies, no setup hassle.

📄

Simple Toolchain

jts to run, jtsc to compile, jtsvm to execute bytecode directly.

Prerequisites

You need Node.js and npm installed on your machine. That's it.

Install Node.js

Download and install Node.js (v14 or later). npm is included automatically.

→ Download Node.js
Note: Node.js 16+ is recommended. LTS versions (18, 20, 22) are ideal.

Verify Node.js and npm

Open a terminal and run these commands to confirm the installation:

shell node --version
npm --version

You should see version numbers like v20.x.x and 10.x.x.

Installation

Install JTS GO globally with a single npm command. Works on Windows, macOS, and Linux.

Install JTS GO

WindowsLinuxmacOS

shell npm install -g @jaytechsolutions/jts-go

Verify Installation

Check that the jts command is available:

shell jts --help

If you see usage info, you're ready to go.

Run Your First Program

Create a file called hello.jts:

jts print("Hello, JTS GO!")

Then run it:

shell jts hello.jts

Output: Hello, JTS GO!

Quick Syntax Guide

JTS GO uses Python-like syntax with end to close blocks.

# Variables
name = "JTS GO"
version = 1.0

# Print
print("Welcome to " + name)

# Conditionals
age = 25
if age >= 18
    print("Adult")
else
    print("Minor")
end

# Loop
for i in 0 to 5
    print(i)
end

# Function
func add(a, b)
    return a + b
end

print(add(3, 4))

Toolchain

JTS GO provides three commands for different workflows:

jts

Run a .jts file directly. Combines compilation and execution in one step.

jts myfile.jts

jtsc

Compile .jts source files to .jtsc bytecode files.

jtsc myfile.jts

jtsvm

Execute compiled .jtsc bytecode files directly on the VM.

jtsvm myfile.jtsc
