Get Started

One command. You're coding.

1

Install Node.js

JTS GO runs on npm. If you don't have Node.js, install it first.

Download Node.js
2

Install JTS GO

One global npm install. No build tools, no configuration.

$ npm install -g @jaytechsolutions/jts-go copy
3

Run Your Code

Create a .jts file and run it instantly.

$ jts hello.jts copy

Features

Built for developers who want simplicity without sacrificing performance.

>/

Python-like Syntax

Indentation-based blocks, dynamic typing, clean expressions. If you know Python, you already know most of JTS GO.

>>

Bytecode VM

Source code compiles to optimized bytecode and runs on a stack-based virtual machine. Fast startup, fast execution.

[ ]

Lists & Collections

First-class list support with literals, indexing, assignment, and append. Mix any types in a single list.

fn

First-class Functions

Define functions with func, pass them around, use recursion. Closures and nested functions supported.

i/o

Smart Input

input() automatically detects numbers, booleans, and strings. No manual type conversion needed.

!

One Command Update

Keep your installation current with jts --update. Always running the latest version.

Language

All the essentials with a clean, minimal syntax.

  • Variables x = 42 — no type declaration needed
  • Strings "hello" + " world" — concatenation with +
  • Conditionals if / else if / else / end
  • Loops for x in 0 to N and while
  • Functions func name(args) ... end
  • Lists [1, 2, 3] with indexing and append
  • Built-ins print, input, len, type, append, number
  • Comments # single line comments

Toolchain

Three commands for every stage of your workflow.

jts

Compile and run a .jts file in one step

jtsc

Compile source to .jbc bytecode

jtsvm

Execute compiled bytecode directly

--update

Update to the latest version via npm
